Visual Studio Code

We will write our code in a computer software called Text-Editor. A fantastic Text-Editor named Visual Studio Code (or VS Code in short) was developed by Microsoft to help software developers write Code.

Click here to see how to install Visual Studio Code in your computer.

VS Code Installations instructions

  1. Click this link to download Visual Studio Code
  2. Click on the Operating System you are using.
  3. Open the downloaded file.
  4. Click "Yes" if any kind of Permission control screen appears
  5. Click Next
  6. Select 'I accept the agreement' and then click Next
  7. Subsequently click on Next and then Next till VS Code gets installed

Git

We will install a special tool in our computer called Git. Its a version management tool that will help us upload and store our code to the internet cloud.

Click here to see how to install Git in your computer.

Git Installations instructions

  1. Click this link to download Git
  2. Click on the Operating System you are using in the Git website.
  3. Open the installer you downloaded.
  4. Select ‘Next’, then in the next screen again select ‘Next’.
  5. Click Next
  6. Then select ‘Use Git from Windows Command Prompt’ radio button and select Next again
  7. Subsequently click on Next and then Install till Git gets installed

7zip

Zip/RAR files are compressed files & folder that are "bundled" together into one single file and take less place. Many of the files are Zipped into .rar, .zip or 7z extension files . To make things easier to Unzip them we will use 7z. Let's install 7z into our System.

Click here to see how to install 7z in Windows.

7z Installations instructions

  1. Click this link to download 7z
  2. Select your Operating System from the website and click Download option.
  3. Open the Downloaded file and click Install
  4. That's it. 7z will automatically installed into your system.

Hyper Terminal

We will now install a fantastic Command Line tool named Hyper. Command Line is something we will be using a lot. All the operating systems have a command line tool of their own.

Click here to see how to install Hyper in Windows.

Hyper Installations instructions

  1. Click this link to download Hyper
  2. Select on Download Hyper for {operating system} option in the Website. This will download Hyper in your Downloads folder.
  3. Then simply run the installation and have Hyper install on your machine.
  4. To actually use Hyper simply right click anywhere on your PC and select Open Hyper Here.

That's it. Hyper is installed into your computer.
