Cmder (Only for Windows)

We will now install a fantastic Command Line tool named Cmder for Windows. Command Line is something we will be using a lot. All the operating systems have a command line tool of their own. To make things easier, we will use a tool for Windows. This is only for Windows User and not for other operating System.

Click here to see how to install Cmder in Windows.

Cmder Installations instructions

  1. Click this link to download Cmder
  2. Select on Download Mini option in the Website.
  3. Right Click on the downloaded file and select 7zip and then select Extract to "cmder_mini"
  4. A folder named cmder_mini will be avilable. We just extracted a Zip file!
  5. Open the cmder_mini folder.
  6. Inside the Cmder folder right click on any file/folder and then select Properties.
  7. Copy the value that is right of Location. Something like "C:\Users\..."
For further instructions go down one more slide.

Adding Cmder to path.

  1. Now close that window and Type with your keyboard: Windows + R together.
  2. This will open the Run window. Type SystemPropertiesAdvanced and then press Enter.
  3. Select Environment Variables.
  4. Now under the "User Variables for %USERNAME%" menu just double click on the Path option.
  5. Select New and then paste the File location you copied earlier.
  6. That's it. Cmder added to PATH
  7. Now go to the cmder_mini folder. Hold Shift and right click
  8. Select Open Command Prompt Here or Open Powershell Here..
  9. Type .\cmder.exe /REGISTER ALL and hit Enter
