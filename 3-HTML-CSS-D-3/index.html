Inline & Block Elements

So, all the elements can be categorized into two more categories based on the width they take up.

Inline

They take up as much space as the elements/text/image/etc.. inside them

Block

They take up the whole 100% width of the parent elements irrespective of the amount of contents inside them.

Understanding how CSS code works


<style>
    body{
        background-color: lightgrey;
    }
</style>

The above code has three parts. Let's see what they are:

Go down
  1. Selector: The body part is the selector. It selects which element we are going to style. Here we are styling body element.
  2. Property: The background-color : part is the property. It actually mentions what we are going to style. Here we are styling the background-color of the body element.
  3. Value: The lightgrey part is the value for the Property background-color. A small : separates property from value. Here we are giving the background-color a value of green.

    4. See that small ; after lightgrey. That is what denotes the end of any CSS property-value pair.

Attributes, Class & Id.

We will finish today's lesson with a tutorial on HTML attributes, class and id.

What are Attributes?

Attributes are like properties of an HTML element like style, size or its identity. Attributes are written like the following inside the opening tag of an HTML element:


    <html lang="en-US">...</html>
    <p title="This paragraph">....</p>
    ....

Here lang="en-US", title="This paragraph" & href="www.google.com" are attributes.

Go down for more about attributes

We will discuss about all the attributes in this course. Let's discuss about the two most used attributes, class & id which are appicable to all the HTML elements.

  1. class: Class is used to group together a particular set of elements that you want to have the same CSS style or any events.
  2. id: Id is used to denote a particular element only.

Explaining more clearly, in an office, comparing all employees, a particular department may be considered a class and an employee of that department may have an id, for example his/her name.
