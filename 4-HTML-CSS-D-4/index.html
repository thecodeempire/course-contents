The CSS Box Model

Before proceeding to add more styles we need to understand the CSS box model. This is quite an important thing when styling HTML elements.

Every HTML element is like a rectangular box. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, & the actual content. Letâ€™s have a look at how this is defined.

Go down for more CSS Box Model knowledge.
  1. Content: In the button that we made the text Button is the content
  2. Padding: Just outside the content is a space called padding.
  3. Border: And around that padding is the border.
  4. Margin: This does not end here. Every element has a margin too outside the element.

Every HTML element follows this box model and carry all these properties.

Adding a Shadow to the element.

Let's add a shadow to the element when it is hovered or clicked.

Go down to know more about box-shadow

The Box Shadow has many values that can be written in one line in one property instead of multiple-lines.

  1. The first value (0.2em) is the horizontal distance of the shadow right from the border of the button element
  2. The second value (0.2em) is the vertical distance below the shadow from the border of the button
  3. The third value(1em) is how much blur you want in the shadow.
  4. And the fourth value (grey) is the color of the shadow

The border can also be written in the same way in a single line. Let's look at that in the slide below.

Go down to change border styles

Let's change the border properties to make it a one liner.


button.btn{
    background-color: darkblue;
    color: white;
    font-size: 3em;
    padding: 0.5em;
    margin: 1em;
    font-weight: bold;
    border-radius: 0.1em;
    border: solid 1em black;
}

We add a new property border and give it 3 values separated by space. Here border-style is the first value, border-width is the second & border-color is the third.

Go down to see the final CSS Code

Adding more than 1 class/id

We can also add more than one class/id in an element. All you have to do is to put the second/third class/id name inside the class attribute with a space in between.

e.g. class="class1 class2"

So, if we add any set of styles to the second/third class they will be added to the already added styles.

And the style that is declared later is always displayed.

HTML Color Codes

Writing Colors in HTML with Color Codes is a difficult thing to remember.

So, there two fantastic ways to write the color we want to display.

  1. Hexa-Decimal Values
  2. As RGB Values

Go down to see what they are

Hexa-Decimal Values

It is written as #rrggbb where rr, gg & bb are color codes of the red, green & blue colors respectively.

Examples of a few colors:

Color Name Color Hexadecimal Code
Black #00000
Red #ff0000
white #ffffff
yellow #ffff00
etc........
Go down to see how RGBA colors are written

Colors can also be represented as separate values of Red, Green, Blue and opacity using rgba(red, green, blue, opacity).

The values of red, blue & green range from 0-255 and of opacity is 0-1.

Color Name Color RBGA Code
Black with full opacity rgba(0,0,0,1)
Red with no opacity (doesn't show) rgba(255, 0, 0, 0 )
White with 0.4 opacity rgba(255,255,255,1)
etc........
